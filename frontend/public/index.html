FastLane

FastLane - Supermarket Navigator

The only real goal of this project is to learn GoogleCloud usage and a lot (a lot, lot, lot) Javascript and Node.JS technologies.

The main porpouse of this page is to have something in frontend and for me to try Angular library. If something works here, it is most likely just consequence. I am using cloud service mostly by curl. :-)

See sources on GitHub.

Frontend service

This page and whole REST API is served by Node.JS application (service) FastLane.Frontend. This service is deployed in Kubernetes container on GoogleCloud. The container is balanced.

Shared data and configuration

Shared data (shared between all users and service instances) are stored in (potentially) replicated Redis instance(s).

Characteristics of these data is, that it is much more often read then write.

Real content consists of: Dynamic configuration, User-list, and other shared content.

User state

Most dynamic data are dedicated per user. These data are managed by custom sharded service. It gave me potential to start more containers when count of actibve users increased.

Custom sharding logic give me control over its strategy. I can distribute users based on access frequency, etc.

Note that this is just demo for me. Policy is primitive (almost none). Service is represented by another Redis instance(s) for now. I just don't want to write another API. :-)
